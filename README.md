# ⚗️ Roopesh Trading Intelligence

A personal options trading tracker and pattern learning system built with React.

## Features
- Live options P&L tracking via Robinhood
- Pattern detection and journal
- Trade history and graduation checklist
- Private algorithm library (stored separately)

## 🔒 Privacy
Pattern algorithms and trading journals are stored in a **separate private repository**.
This public repo contains only the app interface code.
